Rotten potatoes - Documentation

This web application will provide access to information related to movies, movie genres, directors and other movie-related content. A user can sign up, add personal information and create and store a list of movies they deem to be their favourite. The application interacts with a database which stores movie-related data. Below you can find further details on each available endpoint, includng logic, method, a short description, the request format and the response format!

Endpoints:

Logic Method Short Description Request format Response format
'/movies' GET Contains data on all available movies. None Contains all available movies.
/movies/:title' GET Contains data on a single movie using the movie title. None Contains data such as a short description, director, genre and an image URL.
'/movie/:release' GET Contains data on movies by the release year. None Contains data on movies by release year.
'/genre/:name' GET Contains data on movies by genre. None Contains data on movies by genre.
'/directors/:name' GET Contains data about a director. None Contains data about a director.
'/users' POST User registration. Contains data that a new user inputs.

(username: 'Jimmy Jo', password:'jimmyjospw', email:'jimmy@jo.com', dateofbirth:'01.02.1987')

 Contains a text notification that the user has been successfuly added.
'/users/:username/:password/:email/:dateofbirth' PUT Update user information The user can update their username, password, email or date of birth. Text notification that the user information was updated.
'/favorites/:username/:title' POST Adding a movie to the user´s favourite list. None Text notification that the movie has been added to the favourite list.
'/favorites/:username/:title' DELETE Removing a movie from the user´s favourite list. None Text notification that the movie has been removed from the favourite list.
'/users/:username' DELETE User deregistration None Text notification that the user has been deregistered.

Genre Table:

*Descriptions taken from Wikipedia
GenreID Name Description
1 Action An action story is similar to adventure, and the protagonist usually takes a risky turn, which leads to desperate situations (including explosions, fight scenes, daring escapes, etc.). Action and adventure are usually categorized together (sometimes even as "action-adventure") because they have much in common, and many stories fall under both genres simultaneously (for instance, the James Bond series can be classified as both).
2 Adventure An adventure story is about a protagonist who journeys to epic or distant places to accomplish something. It can have many other genre elements included within it, because it is a very open genre. The protagonist has a mission and faces obstacles to get to their destination. Also, adventure stories usually include unknown settings and characters with prized properties or features.
3 Comedy Comedy is a story that tells about a series of funny, or comical events, intended to make the audience laugh. It is a very open genre, and thus crosses over with many other genres on a frequent basis.
